## 🚀 高效控制面板

### 💾 配置三连管家

## 📜 使用指南（极简版）  
```markdown
1. 拖拽灵敏度滑块 → 匹配当前枪械  
2. 游戏内按 [1/2] → 激活1/2号武器配置  
3. 按住 [Q+左键] → 左探头压枪输出  
4. 遇敌按 [Alt+Home] → 秒隐控制界面  
```
