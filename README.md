# 🖱️ 鼠标拖拽控制工具

一款轻量高效的鼠标拖拽增强工具，提供可定制的拖拽速度与多模式触发方式，特别适合需要频繁进行拖拽操作的用户。

## ✨ 核心功能

### 🎚️ 可调拖拽速度
- 可视化滑块控制拖拽灵敏度
- 实时调整，即时生效

### 🎮 多模式触发方式
| 组合键                | 功能描述          |
|-----------------------|-------------------|
| `右键 + 左键`         | 正常压枪      |
| `Q键 + 左键`          | 左探头压枪    |
| `E键 + 左键`          | 右探头压枪    |

### ⚡ 一键显隐
- `Home键`快速切换控制界面显示/隐藏
- 不干扰主工作区
